# FitMoodApp — Додаток для відслідковування спортивних досягнень (JavaFX + SQLite)

Десктопний додаток на JavaFX для ведення щоденника тренувань, перегляду статистики (діаграми/графік), перегляду рекордів та експорту даних у CSV.

## Основний функціонал
- CRUD тренувань: додати / редагувати / видалити
- Таблиця тренувань: дата, тип, тривалість, настрій, коментар
- Дві теми інтерфейсу: світла / темна
- Дашборд зі статистикою (кількість записів, загальна та середня тривалість)
- Діаграма розподілу тренувань за типом
- Графік зміни тривалості тренувань у часі
- Блок “Ваші рекорди”
- Експорт тренувань у CSV (для аналізу/надсилання тренеру, лікарю або реабілітологу)

## Технології
- Java (JDK): встановлена на ПК (запускалось через IntelliJ IDEA)
- JavaFX SDK: **25.0.1**
- База даних: **SQLite** (через `sqlite-jdbc`)
- IDE: **IntelliJ IDEA**

> Це **не web-додаток**, тому порт/браузер (`localhost:8080`) не використовується. Додаток запускається як звичайна програма-вікно.

## Структура проєкту
- `src/Main.java` — точка входу (запуск програми)
- `src/controllers/` — контролери UI (Main/Add/Progress/Splash)
- `src/models/` — моделі даних (наприклад `Training`)
- `src/database/` — робота з SQLite (наприклад `DatabaseHelper`)
- `src/views/` — FXML-вікна (`main.fxml`, `add.fxml`, `progress.fxml`, `splash.fxml`)
- `src/views/styles/` — стилі тем (`light.css`, `dark.css`)

## Як запустити в IntelliJ IDEA (покроково)
1. Відкрити IntelliJ IDEA → **File → Open** → обрати папку проєкту.
2. Переконатися, що обрано JDK:
   - **File → Project Structure → Project → Project SDK**
3. Налаштувати JavaFX у конфігурації запуску:
   - **Run → Edit Configurations…**
   - Обрати конфігурацію запуску класу `Main` (тип *Application*)
   - У поле **VM options** вставити:
--module-path "...\javafx-sdk-25.0.1\lib" --add-modules javafx.controls,javafx.fxml
4. Запустити клас `Main` (кнопка ▶ Run).
5. Додаток відкриється у вигляді вікна.
